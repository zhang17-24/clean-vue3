# 🚀 快速开始 - 5分钟上手

如果你是第一次接触这个项目，按照以下步骤快速开始：

## 第一步：运行项目（2分钟）

```bash
# 在项目目录下运行
python -m http.server 8000

# 或者
npx http-server -p 8000
```

然后在浏览器打开：`http://localhost:8000`

你应该能看到一个计数器，点击按钮数字会变化。

## 第二步：理解入口文件（3分钟）

打开 `src/main.js`，这是整个应用的入口。

### 看这三个部分：

#### 1. 定义状态（setup函数）
```javascript
setup() {
  return {
    count: 0  // 这是响应式数据
  };
}
```

#### 2. 定义UI（render函数）
```javascript
render(state) {
  return h('div', {}, [
    h('button', { onClick: () => state.count++ }, '增加')
  ]);
}
```

#### 3. 挂载应用
```javascript
createApp(App, container);
```

**理解要点**：
- `setup` 返回的数据是响应式的
- `render` 返回虚拟DOM（用JS对象描述UI）
- 点击按钮 → `state.count++` → 自动更新UI
